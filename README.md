# CHETNA
CHETNA (चेतना) - Consciousness That Grows — AI That Learns From Living

What Is CHETNA?
CHETNA is a self-learning AI architecture — a system that genuinely improves from its own experience, without retraining, without human intervention.
Not a model. Not a fine-tuning pipeline. A cognitive growth system that gives any AI agent the ability to:

Learn from outcomes — what worked, what failed, and why
Update its own behaviour — not randomly, but deliberately
Develop preferences — patterns that emerge from repeated experience
Improve over time — every interaction makes it slightly better at being itself

CHETNA is designed to sit on top of TATAV. TATAV gives a being its identity. CHETNA gives that identity the ability to evolve.

The Problem CHETNA Solves
Every AI model you use today was trained, then frozen.
GPT-4 or other llm today cannot learn from your conversation. It will make the same mistake tomorrow that it made today. A robot in a warehouse cannot learn the quirks of your warehouse. A game character cannot adapt to your playstyle.
The gap between knowing and growing — that is what CHETNA fills.

Current AI:
Experience happens → Nothing changes → Same AI tomorrow

CHETNA:
Experience happens
    → Outcome evaluated (good / bad / neutral)
    → Pattern extracted (what caused this outcome?)
    → Behaviour weight updated (do more of this / less of that)
    → Memory enriched (this situation + this outcome = this lesson)
    → Next similar situation handled differently


How CHETNA Works
The Learning Loop
┌───────────────────────────────────────────────── ┐
│              CHETNA LEARNING ENGINE              │
│                                                  │
│   1. EXPERIENCE COLLECTOR                        │

│   2. OUTCOME EVALUATOR                           │

│   3. PATTERN EXTRACTOR                           │
│      What conditions led to this outcome?        │
│      Identifies the variables that mattered      │
│                                                  │
│   4. BEHAVIOUR UPDATER                           │
│      Adjusts weights in the personality engine   │
│      Slowly. Deliberately. With memory of why.   │
│                                                  │
│   5. LESSON MEMORY                               │
│      Stores the learned rule for future recall   │
│      "Low morale + aggressive action = bad"      │
